<p><b>Quaadir Atkinson</b>, professionally known as <b>Neo da Matrix</b>, is an American <a href="/wiki/Hip_hop_production" title="Hip hop production">hip hop producer</a> from <a href="/wiki/Philadelphia,_Pennsylvania" class="mw-redirect" title="Philadelphia, Pennsylvania">Philadelphia, Pennsylvania</a>. He attended Stone Mountain High School in <a href="/wiki/Stone_Mountain,_Georgia" title="Stone Mountain, Georgia">Stone Mountain, Georgia</a>. He has worked with <a href="/wiki/Swizz_Beatz" title="Swizz Beatz">Swizz Beatz</a><sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup><sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup> as an in-house producer for <a href="/wiki/Ruff_Ryders_Entertainment" title="Ruff Ryders Entertainment">Ruff Ryders Entertainment</a> and has produced for the likes of <a href="/wiki/Cassidy_(rapper)" title="Cassidy (rapper)">Cassidy</a>.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup>  Neo Da Matrix is currently working on projects for <a href="/wiki/Cassidy_(rapper)" title="Cassidy (rapper)">Cassidy</a>, <a href="/wiki/Jadakiss" title="Jadakiss">Jadakiss</a>, <a href="/wiki/Swizz_Beatz" title="Swizz Beatz">Swizz Beatz</a>, <a href="/wiki/Young_Chris" title="Young Chris">Young Chris</a>, <a href="/wiki/Eve_(entertainer)" class="mw-redirect" title="Eve (entertainer)">Eve</a>, <a href="/wiki/Fabolous" title="Fabolous">Fabolous</a>, <a href="/wiki/M%C3%BDa" class="mw-redirect" title="Mýa">Mýa</a>, <a href="/wiki/David_Banner" title="David Banner">David Banner</a>, <a href="/wiki/Juelz_Santana" title="Juelz Santana">Juelz Santana</a>, <a href="/wiki/Nina_Sky" title="Nina Sky">Nina Sky</a>, <a href="/wiki/Tru_Life" title="Tru Life">Tru Life</a>, Range and more.
